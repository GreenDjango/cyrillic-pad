# 🗒️ Cyrillic Pad

#### Text Editor for French/Cyrillic language

### 💾 Install

Cyrillic Pad need Qt5

#### 📡 Add Qt5
For ubuntu : `sudo apt install build-essential qt5-default libqt5svg5-dev qtmultimedia5-dev -y`

### 🛠 Build

#### For linux 🐧
`mkdir build && cd build`

`cmake .. && cmake --build .`
